Automate Security with Intelligence

Lucifer bridges the gap between raw hacking tools and agentic AI capabilities.

Async AI Engines

Native integration with Claude 3.5 Sonnet, GPT-4, and local Ollama models. Runs non-blocking reasoning for optimal pentesting guidance.

Terminal Capture

Automatically records shell command history and monitors output buffers. Tracks status to learn from failure and suggest follow-ups.

Vision & OCR Analysis

Scans active application screens and captures visual context. Employs OCR fallback to parse terminal and GUI coordinates headlessly.

Automated Workflows

Built-in steps for Host Discovery, Web App vulnerability scans (Gobuster, Nikto, Sqlmap), and privilege escalation (LinPEAS).

Professional Reports

Compile details, commands run, and screenshots into formatted HTML, Markdown, or JSON reports automatically on teardown.

Dangerous Filters

Audit and filtering capabilities ensure the assistant asks for confirmation before executing commands (like fork-bombs or destructive wipes).

Get Started in Seconds

Simple installation guide to spin up the Lucifer terminal assistant.

# Clone the repository
git clone https://github.com/yashab-cyber/lucifer.git
cd lucifer

# Run the installation script (Kali or Debian-based)
./install.sh

# Activate the environment
source venv/bin/activate
# Edit the environment configuration file
nano .env

# Add your API keys and select provider:
AI_PROVIDER=anthropic
ANTHROPIC_API_KEY=your_key_here
OPENAI_API_KEY=your_key_here

# Test configuration
lucifer config-check
# Run in interactive CLI mode
lucifer start --interactive

# Or run a targeted workflow automatically
lucifer start --target 192.168.1.100 --workflow recon

# Start the backend server for GUI desktop integration
python3 src/lucifer/server.py
